Политика конфиденциальности компании emolokov

Последнее обновление: 26.04.2025

1. Общие положения

Настоящая Политика конфиденциальности регулирует порядок обработки данных при использовании нашего приложения, взаимодействующего с Instagram через официальный API Meta.

2. Какие данные мы используем

Наше приложение работает через официальный API Meta и получает доступ к следующим данным:

  • Публичные данные профиля (через разрешение instagram_basic)
  • Комментарии к Reels (через разрешение instagram_manage_comments)

Важно: Мы не сохраняем эти данные на своих серверах. Все обработка происходит в реальном времени.

3. Как мы используем данные

Полученные данные используются исключительно для:

  • Анализа комментариев к Reels
  • Генерации автоматических ответов на комментарии
  • Обеспечения работоспособности сервиса

4. Интеграция с Meta

Наше приложение использует официальный API Meta в соответствии с Политикой разработчиков Meta. Мы соблюдаем все требования и ограничения, накладываемые Meta.

5. Техническая инфраструктура

Для работы приложения мы используем:

  • Сервис автоматизации n8n, установленный на нашем VPS
  • Хостинг-провайдер Beget (Россия)

  • Доступ к серверам ограничен
  • Регулярное обновление программного обеспечения
